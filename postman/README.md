## Environment variables that need to be created in Postman:

### For all requests

- consumer_key
- consumer_secret
- access_token
- token_secret

### To search for a subscriber by email across all lists OR add a new subscriber in Manage Subscriber

- subscriber_email

### To manage custom fields

- custom_field
- updated_custom_field
